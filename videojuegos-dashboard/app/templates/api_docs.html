GET /api/v1/games

Obtener lista de videojuegos con filtros y paginación

Parámetros Opcionales:

  • platform (string, default: "all") - Filtrar por plataforma
  • genre (string, default: "all") - Filtrar por género
  • publisher (string, default: "all") - Filtrar por editor
  • page (integer, default: 1) - Número de página
  • per_page (integer, default: 50) - Videojuegos por página

Ejemplo de Respuesta:

{
  "status": "success",
  "data": {
    "games": [...],
    "total": 7112,
    "page": 1,
    "per_page": 50,
    "pages": 143
  }
}

GET /api/v1/search

Buscar videojuegos por criterios

Parámetros Requeridos:

  • q (string) - Término de búsqueda

Parámetros Opcionales:

  • type (string, default: "all") - Tipo: platform, genre, publisher, all

Ejemplo de Respuesta:

{
  "status": "success",
  "data": {
    "query": "PS",
    "type": "all",
    "results": [...],
    "total_results": 2
  }
}
